### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git
> pkg install nodejs
> pkg install ffmpeg
> pkg install imagemagick
> git clone https://github.com/MansyurGans/botcokk
> cd botcokk
> npm install
```
###### Run
```bash
> node . [<session name>] (session name is optional)
```
